## Outputs — SQL Query Results

This folder contains 5 Excel files, each storing the output tables generated from the SQL queries used in the EDA analysis.

Each file corresponds directly to one analysis section:
- Section 1 — Portfolio Health & Baseline Risk Profiling  
- Section 2 — Credit Behaviour & Early Risk Signals  
- Section 3 — Grade, Income & Risk Concentration  
- Section 4 — Credit Usage Patterns & Borrowing Behaviour  
- Section 5 — Homeownership, Verification & Stability Indicators  

These files are provided to:
- Show the exact output generated from SQL queries  
- Allow easy review without running SQL code  
- Maintain transparency between SQL logic and analysis insights  

All outputs can be reproduced by running the SQL queries available in the **EDA folder** using MS SQL Server.

# Lending Club Borrower & Credit Risk Analysis

## About This Project
This project is about understanding **borrowers, loans, and credit risk** using a large lending dataset. I wanted to see how different financial factors combine to decide whether someone is **low risk or high risk** for lending.

Instead of just running queries, this project focuses more on **analysing patterns in a very large dataset** and trying to think like a **business or a bank** while looking at the data.

## Problem Statement
The main goal of this project is to answer questions like:

- What factors separate **low-risk borrowers** from **high-risk borrowers**?  
- How do **income levels** interact with **loan grades**?  
- Are there patterns between **credit behaviour** and **loan outcomes**?  
- Can borrower groups be **segmented based on financial behaviour**?

And many more...
